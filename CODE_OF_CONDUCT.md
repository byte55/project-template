# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1, as its Code of Conduct.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: we are committed to providing a friendly, safe, and welcoming environment for
everyone. Be respectful, assume good intent, and keep interactions constructive.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the
project maintainers via GitHub (<https://github.com/byte55>). All reports will be
reviewed and handled confidentially.
